# 公文排版器

## 简单的介绍

这是一款用于公文排版的应用，提供的功能包括：用户选定文种后，按照预定的模板处理用户上传的word文档，或者用户可以直接在线编辑其内容，或调整自动识别的标签。最终生成符合格式要求的word版公文。

## 应用模块结构

应用主要分为四个模块

1. <b>打标签模块。</b>这部分可以用AI，就是给文章标注上面的那些标签。该部分是初步计划使用正则表达式实现。

2. <b>解析模块。</b>根据模块解析标签。该部分已完成实验性代码。

3. <b>前端编辑模块。</b>提供便捷的可视化的编辑方式，方便调整标签。该模块前端界面UI主要参考此前开发的排版应用，但是主要逻辑需要调整，特别是要改为模块化开发，用vue重构。

4. <b>生成模块。</b>根据标签拼接模板生成word。该模块可以复用此前开发的排版应用的部分代码和word模板。
